# 3D Portfolio Project

This is a high-performance 3D portfolio website built with:
- **Vite + React** for the framework.
- **Three.js + React Three Fiber** for the 3D engine.
- **Tailwind CSS** for the UI overlay.
- **Framer Motion** for UI animations.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Features
- **Immersive 3D Experience**: A glowing core with orbiting rings reacting to scroll.
- **Post-Processing**: Bloom and Chromatic Aberration for a premium "Cyber" look.
- **Responsive UI**: Tailwind CSS overlay with your resume details.
- **Interactive**: Scroll down to navigate through Hero, Skills, Projects, and Contact sections.
